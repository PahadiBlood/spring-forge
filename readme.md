# Spring Forge 🚀

**Spring Forge** is a repository dedicated to learning Java and Spring Boot topics in depth.

### 🎯 The Goal
The purpose of this project is to master specific backend concepts by building them hands-on.

To stay efficient, I focus strictly on the **main topic** of each project. This means other parts of the code might not follow production-level standards.

* **Targeted Learning:** If a project is about **Kafka**, I will follow high standards for the Kafka implementation. However, other features like JWT, Security, or Validations will be simplified or ignored to save time.
* **Simplified Logic:** You will see basic implementations for "non-focus" areas (for example, using a User ID as a token) so I can spend more time on the complex architecture.
* **Research focused:** This is a playground for understanding logic and internal workings, rather than building a finished product.
* **Stack:** Java 21 & Spring Boot 4 & PostgreSQL
